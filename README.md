# Smart Attendance System (QR Based)

## Features
- Admin login
- Student add/update/delete
- QR code attendance
- Subject-wise attendance
- Auto date & time
- Present/Absent report
- Mobile scan support

## Tech Stack
- Python (Flask)
- HTML, CSS
- MySQL / SQLite
- QR Code

## How to Run
```bash
pip install -r requirements.txt
python app.py
